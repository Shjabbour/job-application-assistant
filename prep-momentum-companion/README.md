# Momentum Desk

Standalone local motivation companion for job application automation, applications, interview prep, mock interviews, and coding practice.

## Open

Double-click:

```powershell
.\Open Momentum Desk.cmd
```

Or open `index.html` directly in a browser.

## Notes

- No install step is required.
- Data is stored in browser `localStorage` under `prepMomentumCompanion.v1`.
- Use `Reset Today` to clear test progress for the current day.
- Use `Clear All` to wipe the full local app state.
- The app does not call external APIs and does not modify the main job application assistant.
